# Docker DevOps Dashboard

A real-time Docker infrastructure monitoring and container orchestration platform built using React, Node.js, Dockerode, Socket.IO, and MongoDB.
Inspired by modern DevOps platforms like Docker Desktop, Grafana, and Kubernetes observability tools.
# ✨ Features

## 🐳 Docker Management

* List running/stopped containers
* Start / Stop / Restart / Delete containers
* Real-time container monitoring
* Container logs viewer
* Live container stats

## ⚡ Real-Time Infrastructure Monitoring

* WebSocket-powered live updates
* Real-time activity feed
* Live CPU & memory charts
* Docker engine health monitoring
* Animated telemetry widgets

## 🖥️ Advanced Dashboard UI

* Futuristic DevOps-inspired interface
* Glassmorphism + cyberpunk styling
* Responsive monitoring dashboard
* Animated charts and widgets
* Sidebar navigation system

## 🧠 Infrastructure Features

* Event-driven architecture
* Socket.IO real-time communication
* Docker event streaming
* MongoDB integration
* Activity persistence

## 💻 Terminal User Interface (TUI)

* Interactive Docker TUI
* Keyboard-driven container management
* Live logs viewer
* Real-time updates

# 🏗️ Architecture

```text
Frontend (React)
        ↓
Backend API (Express + Socket.IO)
        ↓
Docker Engine (dockerode)
        ↓
MongoDB (event/history storage)
```

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/Himanshu2631/Docker_Devops_Dashboard.git
```
## 2. Install Backend Dependencies

```bash
cd express-backend
npm install
```

## 3. Install Frontend Dependencies

```bash
cd frontend
npm install
```

## 4. Setup Environment Variables

Create `.env` inside backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
```

## 5. Start Backend

```bash
npm run dev
```
## 6. Start Frontend

```bash
npm run dev
```
